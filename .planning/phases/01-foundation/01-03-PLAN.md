  <action>
**Step 1:** Replace entire contents of `client/src/index.css` with:

```css
@import "tailwindcss";

/* Enable class-based dark mode (D-05: dark by default) */
@custom-variant dark (&:where(.dark, .dark *));

/* Dark theme base tokens matching Claude/ChatGPT aesthetic */
@layer base {
  html {
    @apply bg-gray-950 text-gray-100;
  }
}
```

**Step 2:** In `client/index.html`, add `class="dark"` to the `<html>` tag:

Change:
```html
<html lang="en">
```
To:
```html
<html lang="en" class="dark">
```

**Step 3:** Delete `client/src/App.css` (Tailwind replaces all hand-written CSS per D-04):

```bash
rm client/src/App.css
```

Critical Tailwind v4 notes:
- `@import "tailwindcss"` replaces the old `@tailwind base/components/utilities` directives
- `@custom-variant dark` is REQUIRED — without it, `dark:` prefixed classes compile but never activate
- No `tailwind.config.js` or `postcss.config.js` needed — the `@tailwindcss/vite` plugin handles everything
  </action>
